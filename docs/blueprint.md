# **App Name**: SurakshaAI

## Core Features:

- Threat Explainer: Identify potential cybersecurity threats and explain them in clear, accessible language, with references to Indian cyber laws. This feature uses a tool to provide appropriate legal context.
- Mode Detection: Analyze user queries for mentions of cyber threats or legal issues and switch to 'Cybersecurity Threat Mode' automatically.
- Structured Guidance: Present cybersecurity guidance in a structured, easy-to-follow format (Threat/Issue, Steps to Fix, Precautions, Relevant Indian Laws).
- Clean UI Design: Provide a clean, modern user interface with a side navigation bar and a main chat area.
- Tip Saving: Save important tips and resources to a 'Library' for easy access.
- Message Styling: Display AI messages with a light gray background and AI icon, and user messages with a blue bubble and user avatar.
- Link Detection: Automatically warn the user if a phishing or malware link is detected in their message.

## Style Guidelines:

- Primary color: A vibrant teal (#008080) to convey trust and security.
- Background color: A light, desaturated teal (#E0F8F8) to create a calm and secure feel.
- Accent color: A warm yellow (#FFC857) to highlight important information and CTAs.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets and legal references.
- Use shield-themed icons (a custom shield design) to represent cybersecurity concepts and features.
- Implement a side navigation for easy access to different sections (Home, Chat, Library, Search, Settings, Login/Logout).
